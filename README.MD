# Objetivo

El objetivo de esta práctica es diseñar un único prompt siguiendo principios de Prompt Engineering y utilizarlo en diferentes agentes de desarrollo asistido por Inteligencia Artificial para evaluar su capacidad de generación autónoma de software.

La consigna establece que el mismo prompt debe ejecutarse en distintos agentes sin modificar manualmente el código generado, permitiendo comparar resultados y evaluar la calidad de la instrucción diseñada.

# Prompt Utilizado

```text
Actúa como un Diseñador UI/UX de Élite y Desarrollador Frontend Senior. Tu objetivo es diseñar y codificar de forma 100% autónoma una Landing Page premium, ultra-moderna y disruptiva para la empresa de ciberseguridad "CyberTech".

RESTRICCIÓN ABSOLUTA: Entrega todo el proyecto en un único bloque de código HTML independiente y completamente funcional. No utilices marcadores de posición (placeholders) ni omitas partes del código. No modifiques ni rompas las etiquetas indispensables.

TECNOLOGÍAS OBLIGATORIAS:
- Tailwind CSS (vía CDN)
- FontAwesome o Heroicons
- JavaScript nativo

ESTILO VISUAL Y REQUISITOS DE UX PREMIUM:
- Cyberpunk Minimalista / Dark Mode Avanzado
- Fondo #0B0F19
- Tipografías blancas
- Acentos verde neón
- Glassmorphism
- Sombras difusas
- Hover interactivos

ESTRUCTURA DE CONTENIDO REQUERIDA:

1. Cabecera (Header)
2. Hero Section
3. Sobre Nosotros
4. Servicios
5. Testimonios
6. Formulario de Contacto
7. Footer

Optimiza el layout utilizando los prefijos responsive de Tailwind (sm:, md:, lg:) para garantizar una experiencia premium tanto en dispositivos móviles como en pantallas 4K.

Comienza a escribir el código directamente.
```
